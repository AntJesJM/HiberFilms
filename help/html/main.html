Proyecto hibernate

Este proyecto permite el acceso a la base de datos Cine mediante el uso de hibernate.

Dispone de interfaz gráfica y da acceso a tres tablas: Película, Actor y Reparto.En el panel principal estas tablas mostrarán algunos de sus campos, pero no todos.

Sobre estas tablas permite realizar las siguientes acciones:

  • Añadir
  • Borrar
  • Editar
  • Ver
